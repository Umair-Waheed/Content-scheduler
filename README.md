## Content Scheduler App

A full-stack Content Scheduler application built with the MERN stack (MongoDB, Express.js, React.js, Node.js) and JWT-based authentication.

- Registered users can create, schedule, edit, and delete their own posts.

- Styled with Tailwind CSS.

## Features

- User Authentication: Register, Login, Logout (JWT-based)

- Post Management: Create, Schedule, Read, Update, Delete (CRUD)

- Protected Routes: Only owners can edit/delete their posts

- Frontend: React.js with Tailwind CSS

- Backend: Node.js, Express.js, MongoDB

## Setup Instructions

1. Download or clone this project folder.

  ### For Backend

1. Run this on terminal "cd backend"
2. create or add ".env" file in backend folder
3. Run this command "npm install" to install node modules
4. Lastly Run "npm runs server"

By Following these steps backend run successful.

 ### For Frontend

1. Run this on terminal "cd frontend"
3. Run this command "npm install" to install node modules
4. Lastly Run "npm runs dev"

By Following these steps frontend & backend start working.
